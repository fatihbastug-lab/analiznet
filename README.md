# 📊 Ultra Kalite Analiz Pro v3

Bu proje, Excel ve CSV dosyalarındaki verileri kullanarak İstatistiksel Süreç Kontrolü (İSK) yapan modern bir web uygulamasıdır.

## ✨ Özellikler
- **Excel/CSV Desteği:** Tüm hücre formatlarını otomatik algılar.
- **X-Bar Grafikleri:** Ölçüm değerlerini ve kontrol limitlerini ($UCL$, $LCL$) anlık çizer.
- **İstatistiksel Hesaplamalar:** Ortalama ve Standart Sapma hesaplamalarını tarayıcıda yapar.
- **Modern Arayüz:** Tailwind CSS ile responsive tasarım.

## 🚀 Nasıl Kullanılır?
1. Uygulamayı açın.
2. Sağ üstteki 'Dosya Yükle' butonuna tıklayın.
3. Excel dosyanızı seçin ve analiz sonuçlarını anında görün.
